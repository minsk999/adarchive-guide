사내 광고 영상 갤러리

흩어져 있던 광고 영상을
찾고, 재생하고, 가져가세요.

NAS 곳곳의 광고 영상을 한 화면에서. ad.archive를 처음 쓰는 분을 위한 안내서입니다 — 왼쪽 목차에서 필요한 부분으로 바로 이동하세요.

01시작하기

  1. 크롬(Chrome) 등 크로미엄 계열 브라우저로 https://crnas.adef.co.kr:8444 에 접속해요. 즐겨찾기에 넣어두면 편해요.
  2. 구글 로그인 화면이 뜨면 회사 구글 계정(@adef.co.kr)으로 로그인해요. 회사 도메인 계정만 들어올 수 있어요.
  3. 로고 애니메이션이 걷히면 갤러리가 열려요. 첫 접속은 썸네일을 받아오느라 조금 걸릴 수 있어요.
i

한 번 로그인하면 이후엔 바로 열려요. 로그아웃은 설정 → 로그인 계정 → 로그아웃.

02화면 구성

  • 상단 헤더 — 로고, 카테고리 메뉴, 버거 메뉴(외부 링크·설정).
  • 히어로 — 등록된 광고주 이름이 검은 박스 안에서 돌아가요. 아래 숫자는 전체 영상·광고주 수.
  • 필터바 — 검색창, 작업 성격 필터, 저장(내 폴더) 보기. 스크롤해도 따라와요.
  • 갤러리 — 광고주별로 묶인 영상 카드. 최신순이에요.
  • 오른쪽 타임라인 — 연·월을 클릭하면 그 시기로 필터링돼요.
i

마우스 커서가 점·회색 박스로 바뀌는 건 이 사이트의 기본 커서예요. 일반 커서로 쓰고 싶으면 설정 → 라이트 모드를 켜세요.

03원하는 영상 찾기

카테고리

헤더의 카테고리(패션·엔터·식음료/리빙…)를 클릭하면 해당 분야만 보여요. 비딩 카테고리엔 경쟁PT 소재가 따로 모여 있어요.

검색

필터바 검색창에 광고주·캠페인·파일명 아무거나 쳐보세요. 타이핑하는 대로 바로 걸러져요.

작업 성격

필터바의 작업 성격 버튼(▾)을 열면 촬영·모션영상·모션배너·리터치·베리에이션 같은 성격으로 거를 수 있어요.

연·월 타임라인

화면 오른쪽 끝의 연도·월을 클릭하면 그 시기 영상만 남아요. 스크롤 위치에 따라 현재 보고 있는 시기가 자동 표시돼요.

광고주 포커스

갤러리의 광고주 이름을 클릭하면 그 광고주 영상 전체가 한 화면에 모여요. 브랜드가 여러 개면 브랜드별로 다시 묶여요. ← 전체 목록으로 돌아가요.

04카드 다루기

  • 마우스 올리기 — 영상이 소리 없이 미리 재생돼요(GIF는 원본 재생).
  • 클릭 — 플레이어 팝업이 열려요.
  • 우클릭 — 부채꼴 메뉴가 펴져요: 저장 · 복사 · 공유 · 다운로드. 각 버튼을 누르면 선택 팝업이 커서 자리에 떠요.
  • N ver 뱃지 — 같은 작업의 버전(시안→수정본)이 여러 개라는 뜻. 갤러리엔 대표(최신)만 보여요.
i

같은 작업에 영상과 GIF가 함께 있으면 영상본이 대표로 뜨고, GIF는 버전 히스토리에 함께 담겨요.

05플레이어

카드를 클릭하면 열리는 재생 팝업이에요. 재생바·볼륨·전체화면은 유튜브처럼 쓰면 돼요(더블클릭 = 전체화면).

버전 히스토리

버전이 여러 개인 영상은 팝업 옆에 버전 히스토리 레일이 떠요. 클릭하거나 W로 시안→수정본을 오가며 비교할 수 있어요. 레일이 화면보다 길면 그 위에서 휠로 스크롤돼요.

정보와 버튼

제목 아래에 포맷·용량·해상도·길이·fps·수정일이 표시돼요. 하단 버튼으로 저장 · 복사 · 공유 · 다운로드를 바로 할 수 있어요.

06저장 — 나만의 폴더

  1. 카드 우클릭(또는 플레이어의 저장 버튼) → 저장. 처음이면 기본 폴더에 담겨요.
  2. 필터바의 저장을 켜면 내가 담아둔 영상만 모아 볼 수 있어요.
  3. 저장 뷰에서 ＋ 새 폴더로 폴더를 나누고, 이미 저장된 카드는 우클릭 → 저장으로 폴더를 옮겨요.

레퍼런스 모음, 광고주별 제안 준비 등 용도별로 폴더를 나눠 쓰면 편해요. 저장 목록은 내 계정에 묶여 회사·집 어디서든 같아요.

07공유하기 — 두 가지 방법

카드 우클릭 → 공유, 또는 플레이어의 공유 버튼을 누르면 두 가지 중에 골라요.

사내 공유외부 공유 (광고주용)
누가 열 수 있나회사 구글 계정으로 로그인한 사람링크 + 비밀번호를 아는 누구나
로그인필요 (자동)불필요 — 비밀번호만
보이는 정보갤러리와 동일 (경로·버전 전부)영상만 — 내부 정보는 자동으로 가려짐
폴더 공유 시실시간 — 폴더 내용이 바뀌면 링크에도 반영스냅샷 — 만든 순간의 목록으로 고정
유효기간없음설정 가능 (예: 30일) · 폐기 가능
!

공유한 영상의 원본 파일을 NAS에서 옮기거나 이름을 바꾸면 링크에서 그 영상이 조용히 사라져요(재생 불가). 공유 중인 폴더는 정리하지 말고, 정리가 필요하면 링크를 새로 만들어 다시 보내세요.

08외부 공유 — 광고주에게 안전하게

만들기

  1. 카드(또는 저장 폴더)에서 공유 → 외부 공유 (광고주) 선택.
  2. 비밀번호(4자 이상)와 유효기간을 정하고, 필요하면 다운로드 허용을 켜요.
  3. 생성된 팝업에서 링크 칸 클릭 = 새 창에서 열기(미리 확인), 비밀번호 칸 클릭 = 복사예요.
!

아직 재생 호환 변환 전인 소재(.mpg·.mkv 등)를 다운로드 없이 공유하면, 링크를 연 광고주가 재생을 못 할 수 있어요. 이럴 땐 생성 직후 안내가 떠요 — 변환이 끝난 뒤 다시 공유하거나 다운로드 허용으로 발급하세요.

받는 사람에게는 이렇게 보여요

링크를 열면 "외부 공유"라는 비밀번호 입력 화면이 먼저 떠요. 비밀번호를 넣으면 공유한 영상만 깔끔한 갤러리로 보여요. 한 번 입력하면 24시간은 다시 묻지 않아요. 화면 우상단의 라이트 모드 토글로 무거운 효과를 끄고 가볍게 볼 수도 있어요.

자동으로 가려지는 것들

  • NAS 폴더 경로서버 내부 경로가 어디에도 안 나와요
  • 폴더 이름제목·브라우저 탭 모두 회사 브랜드(ADEF.archive)
  • 카테고리·분류사내 분류 체계가 노출되지 않아요
  • 버전 히스토리시안·수정 과정이 보이지 않아요
  • 사내 기능 버튼저장·복사·사내 공유 버튼 없음
  • 계정 정보로그인·이메일 등 내부 정보 없음

지켜지는 보안

  • 비밀번호는 암호화 해시로만 저장돼요 — 관리자도 원문을 볼 수 없어요(잊으면 새로 발급).
  • 비밀번호를 10회 연속 틀리면 15분 잠금돼요.
  • 유효기간이 지나거나 폐기되면 링크는 즉시 열리지 않아요.
  • 누가 언제 열람했는지 기록이 남아요(관리자가 확인).

안전하게 쓰는 요령

  • 링크와 비밀번호는 서로 다른 채널로 보내세요(예: 링크는 메일, 비밀번호는 전화·문자).
  • 유효기간은 필요한 만큼만 — 캠페인 종료일에 맞추는 걸 추천해요.
  • 다운로드 허용은 원본 전달이 꼭 필요할 때만 켜세요.
  • 프로젝트가 끝났거나 잘못 보냈다면 관리자에게 폐기를 요청하세요 — 즉시 차단돼요.

09설정

헤더 버거 메뉴 → 설정. 여기서 바꾼 값은 내 구글 계정에 저장돼 회사·집 PC 어디서든 똑같이 적용돼요(라이트 모드 포함).

  • 재생 설정 — 반복 재생 on/off, 기본 음량.
  • 라이트 모드 — 저사양 PC용. 커스텀 커서·부드러운 스크롤·hover 미리보기 등 무거운 효과를 꺼요.
  • 플레이어 단축키 — 키 배지 클릭 → 새 키 입력으로 변경. 기본값으로 되돌리기도 있어요.

10화면이 버벅일 때

거의 모든 버벅임은 브라우저의 그래픽 가속이 꺼져 있어서예요. 꺼져 있으면 접속 시 화면 위에 안내 배너가 자동으로 떠요.

  1. 배너의 가속 켜는 방법을 눌러 안내창을 열고, 1단계의 [복사] 버튼으로 chrome://settings/system 주소를 복사해요.
  2. 새 탭 주소창에 붙여넣기(Ctrl+V) → Enter로 브라우저 설정 › 시스템으로 이동해요.
  3. 「가능한 경우 그래픽 가속 사용」을 켜고, 재실행을 눌러 브라우저를 다시 열면 끝.
i

가속을 켜도 여전히 버벅이면 배너의 라이트 모드로 보기 또는 설정 → 라이트 모드를 켜세요. 화려한 효과를 끄고 속도에 집중하는 모드예요 — 언제든 다시 끌 수 있어요.

i

화면이 이상하게 깨져 보이면 하드 리프레시: Ctrl+Shift+R.

11모바일에서

  • 검색·필터는 화면 하단 독으로 내려와요.
  • 오른쪽 위 햄버거(≡)를 탭하면 메뉴가 펼쳐지고, 바깥을 탭하면 닫혀요.
  • 카드를 탭하면 플레이어가 열리고, 가로 영상은 전체화면 재생을 지원해요.

12자주 묻는 질문

방금 NAS에 올린 영상이 안 보여요.

새 영상은 자동 스캔(평일 주간 기본 5분 간격)에서 잡혀요. 몇 분 뒤 새로고침해 보세요. 폴더 이름에 '레퍼런스'가 들어가면 스캔에서 제외돼요.

어떤 영상이 재생이 안 돼요.

브라우저가 지원하지 않는 포맷(.mpg·.mkv 등)일 수 있어요. 변환이 완료되면 자동으로 재생되고, 그 전에도 다운로드는 가능해요. 급하면 관리자에게 변환을 요청하세요.

큰 GIF가 한참 뒤에 떠요.

용량이 큰 GIF는 전부 받아야 재생돼서, 그동안 로딩 진행률이 표시돼요. 한 번 본 GIF는 새로고침 전까지 다시 받지 않아 바로 떠요.

보낸 외부 링크가 안 열린대요.

유효기간이 지났거나 폐기됐을 수 있어요. 비밀번호를 10회 틀리면 15분간 잠기기도 해요. 새 링크를 만들어 다시 보내는 게 가장 빨라요.

저장(폴더)이 다른 PC에서도 보이나요?

네. 저장 목록·설정·단축키·라이트 모드 모두 구글 계정에 묶여 있어 어느 PC에서든 같아요.

화면이 하얗게 깨지거나 이상해요.

Ctrl+Shift+R(하드 리프레시)로 새 버전을 강제로 받아오세요. 그래도 이상하면 관리자에게 알려주세요.
